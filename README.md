# Prodizzy

A three-sided marketplace connecting startups, partners, and individuals.

## Features

- **For Startups**: Find investors, partners, and talent
- **For Partners**: Connect with startups seeking your services
- **For Individuals**: Discover opportunities at startups

## Tech Stack

- Frontend: React + Vite + TypeScript
- Backend: Express + Vercel Serverless
- Database: Supabase (PostgreSQL)
- Authentication: Supabase Auth

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Set up environment variables in `.env`

3. Run development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```

## Deployment

Deployed on Vercel with automatic deployments from the `main` branch.
